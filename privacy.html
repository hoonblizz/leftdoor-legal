Privacy Policy

Last updated: April 18, 2026

1. Scope

Leftdoor ("the App") is a single-user personal automation tool that interacts with the TikTok API on behalf of its operator ("the Owner"). This Privacy Policy describes what data the App handles and how. The App does not run a server, does not collect data from third parties, and is not offered as a service to other users.

2. Data the App Handles

When the Owner authorizes the App via TikTok OAuth 2.0, TikTok issues the following credentials, which the App receives and stores locally:

  • access_token — short-lived token authorizing API calls on behalf of the Owner.
  • refresh_token — long-lived token used to obtain new access tokens.
  • open_id — TikTok's hashed identifier for the Owner's account.
  • Token expiration timestamps.

The App also reads metadata about videos the Owner generates locally (title, description, file size, duration) in order to compose upload requests.

3. Where Data Is Stored

All credentials are stored in a single local file on the Owner's machine: .tiktok_tokens.json, with file permissions restricted to the Owner (chmod 600). No credentials, video files, or metadata are transmitted to any server operated by the App's developer. The App has no backend.

4. How Data Is Used

Credentials are used solely to:

  • Authenticate API requests to open.tiktokapis.com.
  • Initialize and complete video uploads via the TikTok Content Posting API.
  • Refresh expired access tokens automatically.

5. Data Sharing

The App does not share any data with third parties. Data flows are limited to:

  • The Owner's local machine (read/write of token file and video files).
  • TikTok's API endpoints (authenticated requests over HTTPS).

6. Retention

Tokens persist locally until they expire or the Owner deletes .tiktok_tokens.json. The refresh token has a finite lifetime defined by TikTok; once it expires, the App requires re-authorization.

7. Your Controls

The Owner can revoke the App's access at any time:

  • In the TikTok mobile app: Profile → Settings and privacy → Security → Manage app permissions.
  • By deleting the local .tiktok_tokens.json file.

Either action immediately ends the App's ability to act on the Owner's behalf.

8. Security

Token files are stored with restricted file system permissions. The App communicates with TikTok over HTTPS. The App does not log credentials.

9. Children's Privacy

The App is not directed to children under 13 (or the equivalent minimum age in the Owner's jurisdiction) and does not knowingly process data from children.

10. Third-Party Services

The App calls APIs operated by TikTok, Google (Gemini), Fal.ai, and Alibaba (Qwen). These providers have their own privacy practices; refer to their respective policies. The App does not transmit TikTok credentials to any of these services.

11. Changes

This Privacy Policy may be updated. The "Last updated" date above will reflect the most recent revision.
